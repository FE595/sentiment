FE 595 Project Summary

Twitter Data - Scraping, sentiment analysis and modelling sentiments

Our objective was to explore different methods of obtaining tweet related data from twitter and to perform analysis on the same. We have used the following methods to obtain data from twitter - Selenium, Tweepy and Old Tweets Libraries.

We have further performed the following actions on the data scraped:

  • Part A - Built a twitter based sentiment indicator of public opinion on the economy (Keyword based)
  • Part B - Built text classification models to identify sentiment of tweets (Opinion Mining)
  • Part C - Tweet based Bitcoin Sentiment Analysis

Part A - Twitter Economy Sentiment Indicator

We have built a public opinion indicator about the US economy using tweets from the past 24 hours.
